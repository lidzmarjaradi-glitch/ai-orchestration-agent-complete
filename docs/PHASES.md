# Product Phases

## Phase 1 — Manual Core Engine
Planner generates executor prompt. User runs it in VS Code Copilot/Claude. Evaluator reviews output.

## Phase 2 — Productized MVP
This package includes the productized manual loop UI.

## Phase 3 — Backend API
Add planner/evaluator API calls from a secure backend.

## Phase 4 — Cloud Product
Add auth, database, project dashboard, team workspace.

## Phase 5 — Automation
Add repo/model integrations and automatic loop execution.

## Phase 6 — Company Layer
Add pricing, onboarding, vertical-specific workflows, and analytics.
